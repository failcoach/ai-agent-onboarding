### What Worked

Be specific. What was useful about this output?

- Did it surface the right things?
- Was the priority order correct?
- Did it save me time vs. doing it myself?
- Was the format easy to scan quickly?

### What Didn't Work

Be specific. What was wrong, missing, or off?

- Was something flagged that shouldn't have been? (False alarm)
- Was something missed that should have been flagged? (Blind spot)
- Was the format hard to use? Too long? Too short?
- Was context missing that I had to add manually?

### Why This Matters

Help the agent understand the reasoning. Examples:

- "The weekly summary led with a project that's actually fine. When the first item isn't urgent, I stop trusting the priority order."
- "The briefing note didn't mention that this client complained last month. Walking in without that context would have been embarrassing."
- "The action items list included tasks that were already done. Outdated items waste my review time."

### What to Do Differently

Clear instruction for next time. Make it specific enough that the agent could follow it without guessing.
