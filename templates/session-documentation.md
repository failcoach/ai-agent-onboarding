# Session Documentation Template

Use this at the end of each work session with your agent. It captures what happened so the next session picks up where you left off. You can fill this in yourself, or ask your agent to fill it in before you wrap up.

Over time, this becomes a running log of your work together. Review it periodically and move lasting decisions into your agent's memory or instructions. For how this fits into the broader memory system, see [guides/03-memory.md](../guides/03-memory.md).
